# Arcee CLI

CLI do Arcee AI para interação com a plataforma.

## Instalação

```bash
pip install -e ".[dev]"
```

## Desenvolvimento

Para executar os checks de desenvolvimento:

```bash
# Formatação
python scripts/format.py

# Lint
python scripts/lint.py

# Testes
pytest
```
python3 -m pip install -e ".[dev]"
